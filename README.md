Real estate price predictor:
This project predicts real estate property using machine learning techniques.
It takes features like Location, area, Number of rooms, and generates accurate price predictions.

Tech Stack:
pythion
pandas, Numpy
Scikit-learn
Flask/ Stramlit
Jupyter Notebook

Model Details:
Algorithm Used - Linear Regression,Desion Tree Regression, Random forest Regression.
Evaluation Metrices - RMSE
Dataset preprocessing steps included- Handeling missing values, Encoding categorical data, Feature scaling.
